# AgroLink+ - Protótipo de MVP

**AgroLink+** é um marketplace inteligente que conecta agricultores e compradores locais, facilitando transações diretas de produtos agrícolas e promovendo uma experiência otimizada para todos os usuários.

## Ideia Central do AgroLink+

- **Compra e Venda**: Facilita transações diretas de produtos agrícolas locais, sem intermediários.
- **Recomendações por IA**: Utiliza inteligência artificial para otimizar preços e prever as melhores decisões de compra e venda.
- **Acesso Direto**: Conecta diretamente agricultores e compradores, proporcionando uma plataforma sem intermediários.

## Telas e Funcionalidades

### Tela de Boas-vindas / Login

- **Nome do App**: AgroLink+
- **Slogan**: "Conectando Agricultores ao Futuro"
- **Botões**:
  - Login com Número de Telefone
  - Cadastrar-se

### Tela de Cadastro

Campos necessários:
- **Nome**: Campo de texto para o usuário
- **Tipo de Usuário**: Agricultor ou Comprador (seleção)
- **Localização**: Entrada de endereço ou GPS
- **Número de Telefone**: Campo para contato
- **Botão principal**: Cadastrar

### Tela Principal (Marketplace)

- **Produtos em Destaque**:
  - Milho Verde - 50 MZN/kg
  - Tomate - 30 MZN/kg
  - Batata Doce - 25 MZN/kg
- **Funções**:
  - Buscar Produto
  - Publicar Produto
  - Ver Clima Local

### Tela de Publicar Produto

Campos necessários:
- **Nome do Produto**
- **Quantidade (kg)**
- **Preço Sugerido (IA)**
- **Foto do Produto**
- **Botão**: Publicar

### Tela de Clima Local

- **Previsão do Tempo**:
  - Hoje: 28°C - Ensolarado
  - Amanhã: 24°C - Chuvas
- **Dica Inteligente**: "Ideal para plantio de milho hoje."
